# @react-google-maps/api example Gatsby.js

![logo](https://raw.githubusercontent.com/JustFly1984/react-google-maps-api/master/logo.png)

## Install

```#!/bin/bash
yarn
```

## Run Development

```#!/bin/bash
yarn start
```

## Build

```#!/bin/bash
yarn build
```

## Clean

```#!/bin/bash
yarn clean
```

## Lint with eslint

```#!/bin/bash
yarn lint
```
